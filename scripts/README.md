# Scripts

このディレクトリには、本プロジェクトのユーティリティ スクリプトが含まれています。以下はそれぞれのスクリプトの説明です。

## generate_conda_file.py

- **目的**: conda の環境ファイルを生成します。
- **使用方法**:
    ```bash
    poetry run python scripts/generate_conda_file.py
    ```
- **出力**: `environment.yml`
